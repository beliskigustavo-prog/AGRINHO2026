Agrofloresta e Sustentabilidade

Site educacional sobre sistemas agroflorestais (SAFs) e meio ambiente, com quiz interativo de avaliação de conhecimento.

Visão geral

Projeto desenvolvido em HTML, CSS e JavaScript puro — sem dependências externas além das fontes do Google Fonts. O objetivo é apresentar de forma acessível e visualmente sóbria os fundamentos da agrofloresta, seus benefícios ambientais e econômicos, além de envolver o usuário por meio de um quiz técnico ao final da página.

Estrutura do projeto

agrofloresta.html — arquivo único com todo o HTML, CSS e JS inline

Conteúdo da página

Seção informativa

Quatro cartões editoriais numerados abordando:

Conceito — o que é agrofloresta e como funciona Ambiente — benefícios ecológicos (só, água, biodiversidade) Economia — projeções produtivas e diversificação de renda Clima — sequestro de carbono e metas climáticas internacionais
